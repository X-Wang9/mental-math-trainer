# Mental Math Trainer

A bilingual mental math practice page for daily arithmetic drills. It includes Chinese and English interfaces, several practice modes, training notes, recent records, and a built-in mental math knowledge library.

Live site: https://x-wang9.github.io/mental-math-trainer/

Category: Brain Health

## Features

- Chinese / English switcher on one page
- Free practice, 10-question rounds, and timed rounds
- Addition, subtraction, multiplication, division, decimals, squares, and 24-point practice
- Training notes before each drill type
- Recent training records, today statistics, current streak, and best streak
- Built-in mental math methods and shortcut library
- Dark theme by default

## Files

- `index.html` - GitHub Pages entry file
- `Mental MathV2.0-bilingual.html` - bilingual wrapper with language switcher
- `Mental MathV2.0.html` - Chinese trainer
- `Mental MathV2.0-en.html` - English trainer

## How To Use

Open the live site:

```text
https://x-wang9.github.io/mental-math-trainer/
```

Or open `Mental MathV2.0-bilingual.html` locally in a browser.

Use the language switcher at the top to choose Chinese or English. Select a practice mode and training type, then click Start.

## Data Storage

Training data is stored locally in the browser with `localStorage`. It does not require a server or account.

The Clear button in recent training records clears:

- Recent records
- Today statistics
- Current streak
- Best streak

Theme and language preferences are kept.
